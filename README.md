## Setup

Clone le repo:

```
git clone https://github.com/hhaidar/flow.git
cd flow
```

Get gulp:

```
npm install -g gulp
```

Party time:

```
npm start
```

## Development

You can build assets by running `gulp`. They are served from `client/build`
